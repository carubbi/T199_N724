# Notebook de Métodos Quantitativos

## Objetivo
Este material apoia a disciplina **Métodos Quantitativos para Computação** com foco em:
- análise exploratória de dados;
- classificação de variáveis qualitativas e quantitativas;
- distribuição de frequências;
- visualizações introdutórias e medidas de tendência central.

## Público-alvo
Alunos de graduação em fase inicial de contato com estatística aplicada e análise de dados em Python.

## Dataset utilizado
- `dataset/api.csv`
- Tema: desempenho estudantil de escolas da Califórnia.
- Estrutura: 5423 linhas e 11 colunas.

## Estrutura didática (resumo)
1. Contexto de análise exploratória de dados.
2. Importação de bibliotecas e configuração do ambiente.
3. Download e descrição do dataset.
4. Carregamento e inspeção inicial dos dados.
5. Classificação de variáveis (qualitativas e quantitativas).
6. Tabelas de frequência e gráficos.
7. Introdução à estatística descritiva univariada e bivariada.

## Alinhamento pedagógico
- Priorizar clareza conceitual antes de automações.
- Manter códigos simples e legíveis para iniciantes.
- Usar exemplos diretamente conectados ao `api.csv`.
- Evitar complexidade desnecessária em visualizações quando o foco for conceito.

## Como executar
1. Criar/ativar ambiente virtual.
2. Instalar dependências:
```bash
pip install -r requirements.txt
```
3. Abrir o notebook:
```bash
jupyter notebook
```
4. Executar as células na ordem.

## Dependências
As dependências estão em `requirements.txt`.
